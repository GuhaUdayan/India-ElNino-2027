Data sources & methodology

IMD — monsoon forecast, block AAS, heatwave data
CWC / WRIS — reservoir storage, 166 reservoirs
CGWB — groundwater block classification
NOAA CPC — Niño 3.4 SST anomaly, El Niño probability 98%
Lin et al. 2023, Geophysical Research Letters — BoB marine heatwave data
INCOIS 2026 — Indian Ocean SST projections
ICAR — crop sensitivity, contingency seed data
CMIP6 ensemble — ±CI confidence intervals
geoBoundaries ADM2 CC-BY 4.0 — India district boundaries
Climate typology model — 6 typologies, district variance
Island connectors — geographic coordinates via D3 Mercator
MGNREGS index — rainfall deficit(35%) + rainfed(30%) + labour(20%) + historic(15%)

All projections are probabilistic. District boundaries shown for reference — data is state-level with typology variance. Island insets are not to scale — crosshairs mark actual geographic positions. All solution suggestions are voluntary and context-dependent. Verify with local administration before implementation.
